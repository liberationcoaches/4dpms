# LCPL Performance Management System
## Project Progress Report

---

## What This System Does

This is a **People Performance Management System** - a web-based application that helps organizations track, measure, and manage employee performance throughout the year.

Think of it as a digital replacement for Excel-based performance appraisals, making the entire process:
- Easier to manage
- More transparent
- Accessible from anywhere
- Automatically calculated

---

## Who Can Use This System?

The system supports **6 different user roles**, each with their own dashboard:

| Role | What They Can Do |
|------|------------------|
| **System Admin** | Manages all organizations, creates reviewers and client admins |
| **Client Admin** | Manages their own organization's setup |
| **Boss (Company Head)** | Oversees all managers, views company-wide performance, sets organizational goals |
| **Manager (Supervisor)** | Manages their team, assigns goals, scores employee performance |
| **Employee** | Views their own goals, submits proof of work, tracks their scores |
| **Reviewer** | External reviewer who validates performance scores |

## Features Completed

### 1. User Registration & Login System
- New users can sign up with email and mobile verification
- Secure OTP (One-Time Password) verification
- Password-protected accounts
- Team code system for employees joining their organization

### 2. Boss Dashboard
- View all managers in the organization
- See company-wide performance statistics
- Assign organizational KRAs (Key Result Areas/Goals)
- Track department-wise performance
- Create and manage managers
- View analytics and reports

### 3. Manager Dashboard  
- View and manage team members (employees)
- Assign work goals (KRAs) to employees
- Score employee performance across 4 review periods
- View team performance metrics
- Track top performers and those needing improvement
- Add mid-cycle notes and feedback

### 4. Employee Dashboard
- View assigned goals and targets
- See performance scores given by manager
- Submit proof of work (links or files)
- Track progress over time
- View historical performance data

### 5. Reviewer Dashboard
- View organizations to review
- Access employee performance data
- Validate and confirm scores

### 6. Admin Dashboard
- Create and manage organizations
- View system-wide analytics
- Manage reviewers and client admins
- Track subscription status

### 7. Performance Scoring System
The system uses a **4-Dimension scoring model**:

| Dimension | Default Weight | What It Measures |
|-----------|---------------|------------------|
| Functional | 60% | Job-specific goals and targets |
| Organizational | 20% | Company-wide objectives |
| Self-Development | 10% | Personal growth and learning |
| Developing Others | 10% | Mentoring and team building |

### 8. Review Cycles
- 4 quarterly review periods per year
- Pilot period for initial setup
- Automated score calculations
- Progress tracking over time

### 9. Notifications System
- Alerts for new assignments
- Reminders for pending reviews
- Updates on score submissions

### 10. Export & Reports
- Export performance data to Excel
- Generate PDF reports
- Department-wise summaries
- Salary hike calculations based on performance

### 11. Profile Management
- Users can update their personal information
- Contact details management
- Password changes

## How It Works (Simple Flow)

```
1. Organization signs up
         ↓
2. Boss creates Managers
         ↓
3. Managers add their Employees
         ↓
4. Goals (KRAs) are assigned
         ↓
5. Employees work on their goals
         ↓
6. Managers score performance quarterly
         ↓
7. System calculates overall scores
         ↓
8. Reports are generated
```
